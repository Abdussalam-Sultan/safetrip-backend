# MINI WHATSAPP CHAT

Want to build mini whatsapp at our Techcrush bootcamp for backend developers

## Requirement

- Authentication we need the following: Bcrypt, -Jsonwebtoken, -expressvalidator, -Middleware
- Email service we need the following: - nodemailer
- logging we need the following: - fs or pino
- Extracting from External API we need the following: -axios and fetch
- Frontend and Backend connection we need the following: -react
- Websocket we need the following: -socket.io, - node http
- caching we need the following: - redis
- Testing we need the following: - jest
- Database we need the following: - sequelize and - mysql2
- Express
- Storage we need the following: - Multer
- dotenv
- Error handling
- RBAC
- Webhook
- TODO push to github



## Installations

- npm install bcryptjs
- npm install jsonwebtoken
- npm install express-validator
- npm install nodemailer
- npm install sequelize
- npm install mysql2
- npm install axios
- npm install ejs
- npm install express
- npm install multer
- npm install dotenv
